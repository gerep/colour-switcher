# Colour Switcher - Agent Guidelines

## Project Overview
**Colour Switcher** is a 2D endless arcade game for browser and mobile platforms.

### Game Mechanics
- **Player**: A horizontal bar at the bottom of the screen controlled by mouse clicks
- **Gameplay**: Colored blobs fall from the top with straight-line trajectories
- **Objective**: Match the player's current color to incoming blobs
- **Scoring**: 1 point per successful color match
- **Failure Condition**: A blob's color doesn't match the player's color when they collide
- **Player Control**: Left mouse click cycles through 4 colors
- **Progression**: Endless game with leaderboard tracking
